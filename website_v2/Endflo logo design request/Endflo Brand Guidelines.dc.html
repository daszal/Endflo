Clearspace
Keep clear space equal to the diameter of the start node (X) on every side. Nothing else — text, edges, other marks — enters this zone.
Minimum size
Lockup: 130px / 28mm wide.
Symbol alone: 24px / 6mm wide.
Color
Ink
#0B0B0C — marks, text
Paper
#F6F5F0 — surfaces
Mist
#DEDCD2 — guides, dividers
Flow Indigo
Links, active states
Signal Amber
Alerts, sparing emphasis
Endflo stays monochrome by default — the mark is always ink or paper, never tinted. The two accents share lightness and chroma and only differ in hue; use them for interactive states and highlights in product UI, never on the logo itself.
Typography
Aa
Manrope · 800
Wordmark and large display moments only — the lockup, section dividers, hero numerals. Always set tight (-4% to -5% tracking).
Aa
Sora · 400 / 600 / 700
Workhorse typeface — headings, UI, body copy. 600–700 for headings, 400 for paragraphs.
Aa
Space Mono · 400 / 700
Technical accents — captions, labels, timestamps, code and log excerpts. Always uppercase with wide tracking when used as a label.
Usage
✓ DO
Use the master file, on plain ink or paper.
✕ DON'T
Don't stretch or distort the proportions.
✕ DON'T
Don't recolor the wave separately from the wordmark.
✕ DON'T
Don't drop it on low-contrast backgrounds or add effects like shadows/glows.
