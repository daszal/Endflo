Endflo's primary interactive control — a pill-shaped button in four variants.

```jsx
<Button variant="primary" size="md" onClick={() => {}}>Run workflow</Button>
```

Variants: `primary` (ink fill, paper text — default CTA), `secondary` (outline), `ghost` (text-only), `accent` (Flow Indigo fill, use sparingly for a single emphasized action). Sizes: sm / md / lg.
